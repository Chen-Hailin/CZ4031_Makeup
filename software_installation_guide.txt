Install Python 3.6, 
$ pip install -r requirements.txt
$ python makeup_test.py
## Should see something like
* Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)

After that, access http://127.0.0.1:5000/ to check the results.

Default running test_example:
					new_sql
					old_sql
					new_qep
					old_qep

new/old_sql is raw SQL text, new/old_qep is JSON format qep, generated by EXPLAIN (VERBOSE, FORMAT JSON) **SQL_QUERY**

To change test_example, change root_path at makeup_test.py. Put new_sql, old_sql, new_qep, old_qep under new root_path. Or change root_path to other test folders provided, such test_example1, test_example2, test_exampleB.
